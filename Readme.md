# Мой проект

Добро пожаловать в мой проект! Это программное обеспечение, разработанное с использованием объектно-ориентированной технологии разработки и современных инструментальных средств.

## Описание

Проект содержит библиотеку `MyLibrary`, которая включает в себя классы `Person` и `Hobby`, а также консольное приложение `ConsoleApp`, которое использует эти классы для проверки их функциональности.

## Функциональность

- Класс `Person` представляет информацию о человеке, включая адрес и список хобби.
- Класс `Hobby` представляет информацию о хобби, включая название и описание.
- Библиотека `MyLibrary` реализует методы для добавления хобби к человеку, отображения информации о человеке и хобби, а также упорядочивания хобби по имени.
- Консольное приложение `ConsoleApp` демонстрирует использование классов из библиотеки `MyLibrary`.

## Использование

1. Склонируйте репозиторий на локальную машину.
2. Откройте решение в вашей среде разработки.
3. Соберите проекты `MyLibrary` и `ConsoleApp`.
4. Запустите консольное приложение `ConsoleApp`.
5. Вы увидите информацию о человеке и его хобби, а также отсортированный список хобби по имени.
